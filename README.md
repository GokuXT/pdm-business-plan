# **Premium Deluxe Motorsport (PDM) \- Business Plan Infographic**

This project is a single-page Interactive Business Plan for the Premium Deluxe Motorsport (PDM) dealership, designed for a FiveM Roleplay environment.

## **Overview**

The business plan is presented as a modern, responsive web infographic. It visualizes key business strategies including:

* **Brand Identity:** "Neon Nightlife" aesthetic tailored to Los Santos.  
* **Market Analysis:** Competitor comparisons and inventory mix.  
* **Financials:** Revenue projections and capital allocation.  
* **Staffing:** Organizational hierarchy and commission structures.  
* **Events:** A roadmap for community engagement.

## **How to View**

### **Option 1: View Live (GitHub Pages)**

If you have deployed this to GitHub Pages:

1. Go to Settings \> Pages in your repository.  
2. Click the link provided (e.g., https://yourusername.github.io/pdm-business-plan/).

### **Option 2: Run Locally**

1. Download or clone this repository.  
2. Open the index.html file in any modern web browser (Chrome, Firefox, Edge).

## **Tech Stack**

This project is built using a single-file architecture for portability:

* **HTML5 & CSS3**  
* **Tailwind CSS (via CDN):** For responsive styling and layout.  
* **Chart.js (via CDN):** For data visualization (Radar, Doughnut, Line, and Bar charts).  
* **Google Fonts:** Using 'Montserrat' and 'Roboto'.

## **Usage in RP**

This document is intended to be used In-Character (IC) as a presentation to:

* City Council / Mayor (for business grants).  
* Investors (for startup capital).  
* Potential Co-owners.
